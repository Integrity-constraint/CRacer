Руководство программиста для CRacer
Введение

CRacer - это простая игра на основе движка Raylib, которая позволяет игроку управлять автомобилем и избегать столкновения с другими автомобилями. Игра состоит из меню и игрового процесса.
Основные компоненты

    Игрок: Игрок управляет своим автомобилем, который может двигаться вправо, влево, вверх и вниз.

    Противники: На экране присутствуют несколько автомобилей противника, которые движутся снизу вверх.

    Таймер: Игра включает таймер, который отсчитывает время, проведенное в игре.

    Музыка: В игре используется музыкальный трек, который воспроизводится в фоновом режиме.

Управление

    Игрок может управлять своим автомобилем с помощью клавиш со стрелками на клавиатуре.

    Для начала игры игрок должен нажать на кнопку "Играть" в меню.

    Для возврата в меню после столкновения с противником игрок должен нажать на кнопку "Назад" в меню.

Код

Все основные функции игры реализованы в коде. Основные функции:

    GetRandomCarPosition: Генерирует случайную позицию для автомобиля противника.

    isColliding: Проверяет, произошло ли столкновение между двумя автомобилями.

    main: Главная функция, которая управляет игровым процессом.
